# SAE3_2025_NGUYEN_CORBILLE_OUMERRETANE



## Introduction

Pour cette première SAÉ de 2ème année dans le parcours Réalisation d’applications, comme le nom le présente si bien, nous devons développer une application qui permet de gérer des informations de stationnement  en voirie à Toulouse. Cette application doit permettre de gérer ces informations aussi bien pour un utilisateur que pour le gestionnaire.

L’application doit fournir : 
Informations claires sur les zones de stationnement
Tarifs
Horaires
Options de paiement
Fonctionnalités pour le paiement et la localisation des places disponibles

Pour cela, on utilisera la méthode SCRUM avec trois sprints principaux. 

Pour l’organisation du développement de cette application nous utilisons GitLab avec une démarche Agile.
